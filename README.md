
# Platformer 2D Prototype

This simple 2D platformer prototype was develop as part of a Unity Developer course, the final goal was to have a simple phase with beginning and end .

# Controls

| Key   | Action
| :---------- | :--------- 
| `Arrow Keys` | `Movement`
| `CRTL` | `Run`
| `Space` | `Jump`
| `S` | `Shoot`
| `P` | `Pause`

## Stack

**Scripts Language:** C#

**Engine:** Unity
